He runs services on a local server with Docker, Traefik reverse proxy, and integrations like Kavita (manga reader) and ntfy (notifications). He's comfortable with both .NET and TypeScript ecosystems, and prefers clean, modern UIs. He values easy plugin development and simple deployment.
